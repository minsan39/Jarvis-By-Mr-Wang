# 系统架构设计

## 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                      Streamlit UI                        │
│                 (聊天界面 + 侧边栏)                        │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                    main.py                               │
│              (UI 入口 + 会话状态管理)                       │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                    src.agent                              │
│                 ┌─────────────────┐                     │
│                 │ get_ai_response │                     │
│                 │     _stream     │                     │
│                 └────────┬────────┘                     │
└──────────────────────────┼──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    LangChain                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐  │
│  │  Messages   │  │   LLM       │  │   Streaming     │  │
│  │  (History)  │  │ (智谱AI)    │  │   Output        │  │
│  └─────────────┘  └─────────────┘  └─────────────────┘  │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                    智谱AI API                             │
│                  (GLM-4 模型调用)                         │
└─────────────────────────────────────────────────────────┘
```

## 模块职责

### 1. main.py
- Streamlit 聊天界面
- 会话状态管理 (`messages`, `current_session`)
- 侧边栏历史会话管理
- 调用 agent 获取 AI 响应

### 2. src.agent.agent
- `create_chat_model()`: 创建智谱AI聊天模型
- `get_ai_response()`: 非流式响应
- `get_ai_response_stream()`: 流式响应生成器

### 3. src.session.session
- `load_session()`: 从文件加载会话
- `save_session()`: 保存会话到文件
- `new_session()`: 创建新会话
- `delete_session()`: 删除会话

## 数据流

1. **用户输入** → main.py
2. **添加到 messages** → st.session_state.messages
3. **调用 get_ai_response_stream** → agent.py
4. **LangChain 组装 messages** → 包含历史记录
5. **流式调用智谱AI** → API 返回 chunk
6. **处理换行符** → `\n` → `<br>`
7. **流式渲染** → st.markdown(response, unsafe_allow_html=True)
8. **保存会话** → 切换会话时保存到 data/ 目录

## 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| LLM | 智谱AI GLM-4 | 国产、成本低、支持流式输出 |
| 框架 | LangChain | 简化 LLM 调用、支持流式 |
| UI | Streamlit | 快速开发、数据可视化友好 |
| 会话存储 | JSON 文件 | 简单、无需数据库 |
