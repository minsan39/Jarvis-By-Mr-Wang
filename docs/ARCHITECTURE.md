## 模块职责

### 1. WeChat Adapter
- 负责与微信客户端交互
- 消息格式转换
- 支持多账号

### 2. Message Router
- 解析用户消息
- 意图识别（闲聊/问答/任务）
- 路由到不同处理模块

### 3. Agent Core
- 协调各组件
- 管理对话状态
- 调用工具链

### 4. LangChain Agent
- 构建Prompt
- 调用智谱AI API
- 解析LLM输出

### 5. RAG Pipeline
- 文档加载 → 文本分割 → 向量嵌入 → 相似度检索
- 支持 PDF/TXT/DOCX

### 6. Memory Management
- 对话历史存储
- 上下文窗口管理
- 关键信息提取

## 数据流

1. **用户消息** → WeChat Adapter
2. **结构化消息** → Message Router
3. **意图+内容** → Agent Core
4. **如果是知识问答** → RAG Pipeline 检索
5. **组装Prompt** → LangChain Agent
6. **LLM生成** → Output Parser
7. **格式化回复** → WeChat Adapter
8. **推送消息** → 用户

## 技术选型理由

| 组件 | 选型 | 理由 |
|------|------|------|
| LLM | 智谱AI GLM-4 | 国产、成本低、支持长上下文 |
| 向量库 | FAISS | Facebook开源、性能高、支持百万级向量 |
| Embedding | text2vec-base-chinese | 中文效果好、模型轻量 |
| 框架 | LangChain | 生态完善、组件丰富、便于迭代 |
| 微信 | wxauto | 纯Python、无需企业微信、可个人使用 |
